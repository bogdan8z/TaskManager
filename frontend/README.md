# Task Manager Dashboard – React + .NET API integration

This folder contains a minimal React (Vite) frontend that talks to the existing .NET backend API.

Quick start

1. Install dependencies:

```bash
cd frontend
npm install
```

2. Start dev server:

```bash
npm run dev
```

Defaults
- The frontend expects the API to be available at `http://localhost:5253`. You can override this by setting `VITE_API_BASE` when running Vite, e.g. `VITE_API_BASE=https://localhost:7240 npm run dev`.
